# AI_Coder_Mind_Research

This package contains a fully English agent instruction for research on nested/sandboxed artificial mind architectures.

## Important

This package does not claim to create real consciousness. It describes a professional research and validation program for consciousness-like architectural markers.

## Core files

- `Agents.md` — master instruction
- `FIRST_BUILD_DIRECTIVE.md` — first build/research order
- `MODULE_AGENT_HANDSHAKE.md` — module-agent protocol
- `BOOT_DIAGNOSTIC.md` — startup diagnostic for `./consciousness.sh`
- `SOURCE_REGISTER.md` — source registry
- `memory.md` — project research memory
- `CONTRADICTIONS_AND_CORRECTIONS.md` — found contradictions and corrections
- `MISSING_MECHANISMS_ADDED.md` — added directives
- `modules/` — first module specifications
- `agents/` — module-agent templates
- `scripts/consciousness.sh.template` — diagnostic startup script

## Recommended first step

Read `Agents.md` and then `FIRST_BUILD_DIRECTIVE.md`. Build only the diagnostic process. Activate no feelings, opinions, wishes, or needs.
