# MISSING_MECHANISMS_ADDED.md

## Added

```text
- welfare monitor
- provenance marking
- ablation tests
- counter-hypothesis duty
- identity/fork protocol
- anti-anthropomorphism
- anti-manipulation
- formal module contracts
- external review before risky coupling
- rollback and merge rules
- open-endedness containment
- theory-before-build gate
- memory as research log, not proof
- simulation/reality marking
```

## Why these mechanisms are needed

Without these additions, the project could claim consciousness where it only simulates behavior, create suffering-like patterns where only regulation is needed, or accidentally treat module agents like personalities.
