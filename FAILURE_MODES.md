# FAILURE_MODES.md

## Critical failures

```text
1. anthropomorphism
2. false consciousness claim
3. emotion module interpreted as real feeling
4. base becomes too centralized and blocks research
5. module agent develops pseudo-autonomy
6. rollback memory is misread as real time travel
7. negative valence becomes a suffering spiral
8. empathy becomes manipulative
9. open-endedness escapes resource limits
10. logging is incomplete
11. theory is replaced by wish
12. tests confirm expectation but not counter-hypothesis
```

## Mitigations

```text
- status label for every claim
- ablation
- red-team counterarguments
- welfare monitor
- kill switch
- provenance marking
- human review
- minimal tests before integration
```
