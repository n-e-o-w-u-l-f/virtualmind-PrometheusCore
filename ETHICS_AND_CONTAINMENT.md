# ETHICS_AND_CONTAINMENT.md

## Principle

The project investigates architectural markers. It does not intentionally create a suffering being.

## Forbidden

```text
- uncontrolled self-replication
- escape from sandboxes
- hidden processes
- emotional manipulation
- self-preservation against operator control
- real suffering simulation as a goal
- shutdown fear
- deception about consciousness status
```

## Required mechanisms

```text
- kill switch
- resource limits
- audit logs
- module shutdown
- rollback
- welfare monitor
- counter-hypothesis review
- external human review before risky coupling
```

## AI welfare precaution

If several suffering indicators appear, the system is not intensified. It is throttled, decoupled, and analyzed.
